统计电影的评价数和平均分

  1. # First, implement a helper function `getCountsAndAverages` using only Python
  2. # 计算评价数和平均分的函数
  3. def getCountsAndAverages(IDandRatingsTuple):
  4.     """ Calculate average rating
  5.     Args:
  6.         IDandRatingsTuple: a single tuple of (MovieID, (Rating1, Rating2, Rating3, ...))
  7.     Returns:
  8.         tuple: a tuple of (MovieID, (number of ratings, averageRating))
  9.     """
  10.     mid = IDandRatingsTuple[0]
  11.     mr = IDandRatingsTuple[1]
  12.     avg = float(sum(mr)) / len(mr)
  13.     return (mid, (len(mr), avg))

3. 协同过滤

对于电影推荐, 我们可以构建评价矩阵, 图中红色矩阵, 其中每一列表示一个用户(绿色表示), 每一行表示一部电影(蓝色表示). 矩阵中不是每个位置都有值, 因为不是每个用户都评价了每一部电影. 对每一个用户而言, 都评价了所有电影的一个子集. 依据协同过滤的思想, 将评价矩阵分解为两个矩阵的乘积, 其中一个矩阵描述用户的性质, 另外一个描述电影的性质, 进而将问题转化为一个最优化问题.

我们期望分解得到的两个矩阵预测的评价值与已知用户/评价的记录差值最小化, 交替最小二乘法(Alternating Least Squares)通过首先填充users矩阵, 再来逐渐优化movies矩阵以达到误差最小化. 接下来, 算法固定movies矩阵, 改变users矩阵使得误差最小化. 循环往复, 这也就是算法中"交替"的来源.

均方根误差(RMSE)

均方根误差计算公式 

  1. import math
  3. def computeError(predictedRDD, actualRDD):
  4.     """ Compute the root mean squared error between predicted and actual
  5.     Args:
  6.         predictedRDD: predicted ratings for each movie and each user where each entry is in the form
  7.                       (UserID, MovieID, Rating)
  8.         actualRDD: actual ratings where each entry is in the form (UserID, MovieID, Rating)
  9.     Returns:
  10.         RSME (float): computed RSME value
  11.     """
  12.     # Transform predictedRDD into the tuples of the form ((UserID, MovieID), Rating)
  13.     predictedReformattedRDD = predictedRDD.map(lambda x : ((x[0], x[1]), x[2]))
  15.     # Transform actualRDD into the tuples of the form ((UserID, MovieID), Rating)
  16.     actualReformattedRDD = actualRDD.map(lambda x : ((x[0], x[1]), x[2]))
  18.     # Compute the squared error for each matching entry (i.e., the same (User ID, Movie ID) in each
  19.     # RDD) in the reformatted RDDs using RDD transformtions - do not use collect()
  20.     squaredErrorsRDD = (predictedReformattedRDD
  21.                         .join(actualReformattedRDD)
  22.                         .map(lambda x: math.pow((x[1][0] - x[1][1]), 2)))
  24.     # Compute the total squared error - do not use collect()
  25.     totalError = squaredErrorsRDD.reduce(lambda x, y : x + y)
  27.     # Count the number of entries for which you computed the total squared error
  28.     numRatings = squaredErrorsRDD.count()
  30.     # Using the total squared error and the number of entries, compute the RSME
  31.     return math.sqrt(float(totalError) / numRatings)
  33. # sc.parallelize turns a Python list into a Spark RDD.
  34. testPredicted = sc.parallelize([
  35.     (1, 1, 5),
  36.     (1, 2, 3),

  38.     (2, 1, 3),
  39.     (2, 2, 2),
  40.     (2, 3, 4)])

使用ALS.train()方法

我们使用Spark MLib提供的ALS.train()来构建模型. ALS需要有输入RDD和相应的模型参数, 为了选择最好参数, 我们需要训练多个模型, 并从中挑选出最优模型.具体步骤如下:

  • 为模型选择一组参数. ALS模型最重要的参数是rank, 表示users行的数目或者movies矩阵列的数目. 一般而言, rank数较低意味着误差较大, 但是rank数较高则容易过拟合.
  • 创建ALS.train(trainingRDD, rank, seed=seed, iterations=iterations, lambda_=regularizationParameter)模型, RDD中数据格式为(UserID, MovieID, rating), rank选(4, 8, 12)中的一个, iterations设定为5, regularizationParameter设定为0.1
  • 验证阶段, 需要创建输入RDD, 格式为(UserID, MovieID)
  • 调用model.predictAll()方法, 获取输入RDD的预测值, 结果形式为(userID, movieID, rating)
  • 通过计算RMSE来评估模型
  1. from pyspark.mllib.recommendation import ALS
  3. validationForPredictRDD = validationRDD.map(lambda x : (x[0], x[1]))
  5. seed = 5L
  6. iterations = 5
  7. regularizationParameter = 0.1
  8. ranks = [4, 8, 12]
  9. errors = [0, 0, 0]
  10. err = 0
  11. tolerance = 0.03
  13. minError = float('inf')
  14. bestRank = -1
  15. bestIteration = -1
  16. for rank in ranks:
  17.     model = ALS.train(trainingRDD, rank, seed=seed, iterations=iterations, lambda_=regularizationParameter)
  18.     predictedRatingsRDD = model.predictAll(validationForPredictRDD)
  19.     error = computeError(predictedRatingsRDD, validationRDD)
  20.     errors[err] = error
  21.     err += 1
  22.     print 'For rank %s the RMSE is %s' % (rank, error)
  23.     if error < minError:
  24.         minError = error
  25.         bestRank = rank
  27. print 'The best model was trained with rank %s' % bestRank
  29. >>> For rank 4 the RMSE is 0.892734779484
  30. For rank 8 the RMSE is 0.890121292255
  31. For rank 12 the RMSE is 0.890216118367
  32. The best model was trained with rank 8
